# PirateGame
A modification of 2025 AQA AS Computer Science code for exam with fixed bugs and added new features in gameplay. 

New features include: 
    1. Inability to dig the same place twice and leaving dig holes on those places
    2. Dynamite that can be used by pressing B
    3. Rising water level, in which the pirate can be drowned
    4. Moving tresure
    5. Colourful text
    6. Giant moves
    7. Inventory with ability to leave coconuts near hut for additional score
    8. Scanner that can be used through Inventory
    9. Writing what items the pirate missed at the end
    10. Auto generated map that is shown in real time
    11. Different levels of difficulty
